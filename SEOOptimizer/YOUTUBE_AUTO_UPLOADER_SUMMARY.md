## 🚀 **How to Use**

### **Quick Start**
```bash
cd "c:\Users\brook\OneDrive\Documents\Scripts\SEOOptimizer"
python youtube_auto_uploader.py
```

### **Step-by-Step Workflow**

1. **📁 Video Path**: Point to your video file
   ```
   📁 Enter path to video file: C:\Videos\my_sniper_short.mp4
   ```

2. **🖼️ Thumbnail** (regular videos only): 
   ```
   🖼️ Enter path to thumbnail: C:\Thumbnails\sniper_thumb.jpg
   ```

3. **📋 Playlists**: Choose existing playlists
   ```
   📝 Enter playlist names: Into the Radius 2, VR Shorts
   ```

4. **✍️ Describe Your Content**:
   ```
   💭 Your description: I did some incredible long-range sniper shots 
   in Into the Radius 2 using the new SR-25 rifle with the improved 
   scope from the v0.14.1 beta update
   ```

5. **🤖 AI Generates Everything**:
   - **Title**: "INSANE VR Sniper Shots! 🎯 Into the Radius 2 SR-25 #vr #sniper #intotheradius2 #shorts"
   - **Description**: Full optimized description with your successful patterns
   - **Tags**: All your proven high-performing tags

6. **✅ Review & Approve**:
   - See generated content
   - Approve (y), Cancel (n), or Edit (e)

7. **🚀 Upload Complete**:
   - Video uploaded with all metadata
   - Thumbnail set (if provided)
   - Added to playlists
   - Returns YouTube URL
